## Raw Data

The `data/raw` directory contains the unprocessed transaction data used in this
project.  The main file is `SAML-D_reduced.txt`, which is a reduced version
of the SAML‑D dataset.  This file includes sensitive financial information
and **must not be shared publicly**.  Please treat the raw data with
confidentiality and comply with any applicable data‑handling policies.

Any additional raw files should be placed in this directory with a brief
description added to this README.
