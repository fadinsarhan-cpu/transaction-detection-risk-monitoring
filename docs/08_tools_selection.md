# Tools Research and Selection

Our technology stack was chosen to balance flexibility, scalability and ease of use:

- **Python** and **Jupyter notebooks** for data preparation, exploration and modelling. The rich ecosystem of libraries (pandas, scikit‑learn, XGBoost, imbalanced‑learn) facilitated rapid prototyping.

- **Power BI** for dashboarding. It offers interactive visuals, slicers and drill‑through that are well suited to compliance analysts.

- **GitHub** for version control and collaboration. Using a template repository ensured that our documentation is organised and easy to navigate.

- **Kaggle API** for data acquisition. It provides a straightforward way to download the dataset and integrate updates.

Alternative tools, such as R or Tableau, were considered; however, Python and Power BI provided sufficient capabilities for the scope of this project.
