# Dashboard Screenshots

This folder should contain PNG images of each page of the Power BI
dashboard.  These screenshots are required to evidence the visual design
and functionality of the dashboard for submission.

Please add the following files before final submission:

1. **01_executive_overview.png** – Overview page showing KPI cards, transaction trend and summary statistics.
2. **02_transaction_risk.png** – Transaction risk monitoring page with charts for payment type, currency, banks and time.
3. **03_high_risk_accounts.png** – High‑risk accounts page with top senders/receivers and risk bands.
4. **04_model_performance.png** – Model performance page illustrating confusion matrix, evaluation metrics and model comparison.

If these screenshots are not yet available, run your Power BI dashboard
using `Dashboard_Fadi.pbix` and export the required pages as PNG files.
