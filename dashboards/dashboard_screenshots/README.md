# Dashboard Screenshots

This folder should contain the exported images of the Power BI dashboard pages for final submission. Please export the screenshots from your Power BI file and save them here with the following file names:

- **01_executive_overview.png** – An overview of KPIs, suspicious transaction rate, and trends to summarise overall risk.
- **02_transaction_risk.png** – Visuals for suspicious transactions broken down by payment type, currency, bank and time.
- **03_high_risk_accounts.png** – Bar charts or tables highlighting top sender and receiver accounts with high risk scores or suspicious ratios.
- **04_model_performance.png** – Model evaluation visuals, such as the confusion matrix, precision-recall table and ROC curve.

These screenshots must be exported from `dashboards/Dashboard_Fadi.pbix` using Power BI before submitting the project. If the PBIX file is not part of the repository, please generate the dashboard externally, take screenshots as specified, and place them in this folder.
