# Model Result Images

This folder should contain visualisations exported from `notebooks/04_machine_learning.ipynb` after you run the notebook on the cleaned dataset.

- **confusion_matrix.png** – a heatmap showing counts of true positives, false positives, true negatives and false negatives for the selected model.
- **roc_curve.png** – a receiver operating characteristic curve plotting true positive rate against false positive rate across different thresholds.
- **feature_importance.png** – a bar chart showing the relative importance of the top predictive features (for tree‑based models such as Random Forest or XGBoost).

If these images cannot be generated automatically, save them manually from the notebook and place them in this folder before final submission.
